# everyday_github

一个用于每日追踪 AI 开源生态与社交平台热点的知识仓库。

## 项目定位

- 跟踪 GitHub 每日高热 AI 项目。
- 为新上榜项目建立中文解读目录（含用法、原理、风险和补充建议）。
- 生成跨平台 AI 热点日报（GitHub、X、Instagram、YouTube）。

## 使用方式

1. 在仓库根目录执行每日更新流程。
2. 新增当天目录：`daily/YYYY-MM-DD/`。
3. 写入 `ai-hotspots.md` 并更新当日新项目或高热项目目录说明。

## 目录结构

- `daily/`：按日期归档的热点日报。
- `<project>/README.md`：单个 AI 项目的中文介绍与分析。
- `AGENTS.md`：自动化执行约束。
- `TODO.md`：待办与阶段记录。

## 重要说明

- 默认直接在当前分支更新（不新建分支）。
- 热点内容优先采用官方来源与主流媒体交叉验证。
- 社媒互动计数不可稳定抓取时，采用“连续报道密度 + 官方动作强度”作为替代信号。
- 若 GitHub 头部热点已在仓库中建立目录，则优先刷新既有 README，并在当日日报中明确记录 `新建目录：无`。
- 若仓库名存在同名冲突或已有不同项目占用目录，优先避免覆盖既有目录，并在日报中显式说明取舍。
