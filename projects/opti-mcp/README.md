# OptiMCP

## 定位

`OptiMCP` 是面向 agent 数学与逻辑结论的确定性验证 MCP：按用户规则检查结果，并指出违反的规则。

截至 2026-07-23，GitHub API 显示其创建于 2026-07-21，约 7 stars；属早期开发者信号。

## 用法

将其 MCP server 接入支持 MCP 的 agent，并把业务约束、公式或判断规则作为可验证输入，在模型输出后请求校验。

## 原理

模型负责生成候选答案，规则引擎以确定性计算复核数值和逻辑，从而将“可信”落到明确的失败规则上。

## 价值

- 为报价、配置、工程计算等高约束任务增加可审计的 guardrail。
- 可减少仅靠提示词约束的幻觉数值。

## 风险边界

- 规则覆盖不等于业务正确；遗漏的规则无法被发现。
- MCP 接入需要评估输入数据、日志和权限暴露面。

## 补充建议

先把关键不变量转成单元测试或规则，再让 agent 调用；不要以工具名称替代独立复算。
