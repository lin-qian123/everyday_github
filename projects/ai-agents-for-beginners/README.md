# ai-agents-for-beginners

## 定位
`microsoft/ai-agents-for-beginners` 是面向入门者的 Agent 教程仓库，以课程化结构降低上手门槛。

## 用法
1. 按课程顺序学习基础概念与示例。
2. 在本地复现实验并记录每节课的可迁移模式。
3. 将示例改造为你的业务场景（工具调用、记忆、评估）。

## 原理
- 通过分课模块覆盖 Agent 构建核心环节。
- 使用可执行 Notebook/示例代码把抽象概念落地。
- 以渐进式任务降低学习曲线。

## 价值
- 对新成员培训效率高。
- 便于作为团队知识共享材料。
- 可作为内部 POC 的教学脚手架。

## 风险边界
- 教学示例通常弱化生产复杂度。
- 版本更新快，旧教程可能与新 API 不一致。
- 不应直接照搬到高风险业务流程。

## 补充建议
- 学习时同步记录“哪些内容已过时”。
- 为每课补一条“生产化改造建议”。
- 加入评测与成本监控作为毕业动作。
