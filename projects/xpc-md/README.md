# XPC.md

## 定位

`XPC.md` 是 Markdown-only 的 prompt 框架，目标是为不同 AI agent 建立持久项目记忆，减少跨会话反复解释。

截至 2026-07-23，GitHub API 显示其创建于 2026-07-22，约 1 star；属早期开发者信号。

## 用法

将框架文件放入仓库并按模板记录项目背景、决策、约束与当前状态，再由宿主 agent 在任务开始时读取。

## 原理

使用版本控制友好的文本文件作为显式共享状态，避免将关键上下文锁在某个模型供应商的会话中。

## 价值

- 轻量、可审查，适合已有 Markdown 工作流的团队。
- 有助于把长期项目的决策和接续线索放进仓库。

## 风险边界

- 文本记忆会陈旧，且不能自动保证被模型正确遵守。
- 不应把密钥、个人数据或不必要的聊天记录写入仓库。

## 补充建议

配合简洁的状态更新制度和验证记录，避免把文件膨胀成不可读的聊天转储。
