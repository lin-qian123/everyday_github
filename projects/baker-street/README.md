# Baker-Street

## 定位

`Baker-Street` 是面向 Claude Code 的多视角分析框架，用七种认知原型并行发现论证冲突与盲区。

截至 2026-07-23，GitHub API 显示其创建于 2026-07-21，约 3 stars；属早期开发者信号。

## 用法

将框架作为复杂决策、设计审查或研究任务的第二意见流程，并要求输出可回溯到证据的结论。

## 原理

通过分配相互不同的分析角色，强制生成替代假设和反例，避免单一 agent 轨迹过早收敛。

## 价值

- 适合架构权衡、风险评审和研究计划的碰撞式思考。
- 可与独立验证器形成“提出假设—找反例—落证据”的闭环。

## 风险边界

- 多角色文本不等于独立事实来源。
- 并行推理会增加 token 与协调成本。

## 补充建议

为每个视角设定证据要求，并让最终决策者显式处理冲突而非投票取平均。
