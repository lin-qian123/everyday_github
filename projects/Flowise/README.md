# Flowise

## 定位
`FlowiseAI/Flowise` 是面向低代码/可视化编排的 LLM 应用构建平台，适合快速搭建聊天、检索问答和代理流程。

## 用法
1. 按文档启动 Flowise（Docker 或 Node 环境）。
2. 在可视化画布中拖拽组件，配置模型、检索与工具节点。
3. 通过 API 或嵌入方式把流程接入业务系统。
4. 对关键流程加上鉴权、限流、日志与回放。

## 原理
- 把 LLM 应用抽象为节点图（模型、提示词、向量库、工具）。
- 节点间通过数据流连接，形成可调试的工作流 DAG。
- 结合向量数据库与插件实现 RAG 与工具增强。

## 价值
- 降低 AI 应用原型开发门槛。
- 可视化有助于跨职能协作（产品/工程/运营）。
- 便于快速试错和流程复用。

## 风险边界
- 可视化流程在复杂场景下可能出现配置漂移。
- 低代码平台仍需工程化治理（版本、测试、监控）。
- 若缺少权限隔离，外部工具调用风险较高。

## 补充建议
- 把“可视化原型”与“生产编排”分层管理。
- 为关键节点建立回归测试样本集。
- 明确提示词、知识库、工具配置的变更审计。
