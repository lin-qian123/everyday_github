# repro

## 定位

`repro` 是 Claude Code skill，目标是将“偶发崩溃”收敛为最小可复现脚本，并交付回归测试。

截至 2026-07-23，GitHub API 显示其创建于 2026-07-22，约 1 star；属早期开发者信号。

## 用法

提供错误现象和可访问环境，让 skill 逐步缩小触发条件；最终将最小复现和测试纳入仓库。

## 原理

遵循假设、收紧触发器、delta debugging、固定回归测试的路径，把模糊失败转化为稳定证据。

## 价值

- 与只会重试或加 sleep 的 agent 调试方式形成对照。
- 产物可持续保护后续改动不重新引入缺陷。

## 风险边界

- 最小复现可能需要敏感数据或生产条件，必须做脱敏与隔离。
- 随机性、并发和外部服务依赖不一定能完全收敛。

## 补充建议

要求每次推断都有对照实验，且把复现脚本固定到 CI 可运行条件。
