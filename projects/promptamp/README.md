# promptamp

## 定位

`promptamp` 是浏览器扩展式的 BYOK prompt 增强器，面向 Chrome、Firefox、Edge 中的任意输入框。

截至 2026-07-23，GitHub API 显示其创建于 2026-07-22，约 1 star；属早期开发者信号。

## 用法

按浏览器扩展说明安装并配置自有模型密钥，在发送前触发 prompt 改写。

## 原理

在网页输入层截获文本，调用用户配置的模型生成更具体的提示，再回填原输入框。

## 价值

- 把 prompt 辅助从单一 AI 产品扩展到跨站点工作流。
- BYOK 允许用户选择模型和计费渠道。

## 风险边界

- 扩展可读取网页输入，必须检查权限、网络请求和密钥存储。
- 自动改写可能改变用户意图或泄露敏感文本。

## 补充建议

仅在非敏感站点试用，默认保留人工确认和原始文本对比。
