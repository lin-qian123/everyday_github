# OpenClaw 今日笔记（2026-02-05）

定位
OpenClaw 是“本地优先 + 多渠道聊天入口 + 可扩展工具”的个人 AI 代理平台。它把聊天应用当作统一入口，通过本地控制平面调度模型与工具完成任务。

为什么今天值得关注
- GitHub 今日趋势靠前，属于近期关注度高的 AI 工具之一。
- 媒体对其热度与潜在安全风险有集中报道，说明“本地 AI + 系统权限”是用户关切点。

上手建议
- 先以最小权限运行：只接入一个聊天通道 + 只开放必需工具。
- 完成任务后逐步打开更多工具权限，避免一次性暴露全部系统能力。
- 公开访问必须放在反向代理 + TLS 后面，避免直接暴露网关。

原理简图
用户（聊天入口） -> 本地控制平面（编排） -> 工具/技能（执行） -> 回写结果
