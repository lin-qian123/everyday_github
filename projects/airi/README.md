# airi

## 项目定位
`airi` 是一个可自托管的 AI Companion 项目，强调“用户可拥有、可扩展”的实时语音与虚拟角色交互能力，并支持 Web/macOS/Windows。

- GitHub: https://github.com/moeru-ai/airi
- 文档站点: https://airi.moeru.ai/docs/
- 热度参考：GitHub Trending（2026-05-26 抓取）显示该仓库总星标约 39.7k。

## 用法

1. 按官方文档准备运行环境和依赖。
2. 先使用默认配置跑通基础会话或语音链路。
3. 结合扩展模块（语音、游戏连接器等）做个性化功能增强。

## 原理

- **实时交互链路**：通过语音输入、识别、语言生成、语音输出组成低延迟回路。
- **角色与外观系统**：整合 Live2D/VRM 等资产，让交互具有“数字角色”表达。
- **插件化扩展**：可接入外部模块（示例包括游戏连接能力），形成可组合生态。

## 价值

- 对虚拟陪伴、直播互动、教育演示等场景有较强可玩性。
- 自托管能力使用户在数据与部署上有更高控制权。
- 适合作为“多模态实时 Agent”实验平台。

## 风险边界

- 该类产品涉及人格化交互，需重视伦理、内容安全与误导风险。
- 实时语音链路对算力和网络稳定性要求较高。
- 若引入第三方插件，需额外审查安全与权限边界。

## 补充建议

- 建议先以离线或受控测试环境验证稳定性与安全策略。
- 明确用户告知：系统输出不构成专业建议，避免过度拟人化误解。
- 为插件和模型调用建立审计日志，便于问题追踪。
