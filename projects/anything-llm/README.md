# anything-llm

## 定位
`Mintplex-Labs/anything-llm` 是面向团队知识问答与私有部署的全栈 LLM 工作台，强调“文档接入 + 向量检索 + 多模型推理 + 权限管理”的一体化体验。

## 用法
1. 按官方部署说明启动服务（本地或服务器）。
2. 接入模型提供方与 Embedding/向量存储组件。
3. 导入知识源（文档、网页、仓库内容）并建立 workspace。
4. 配置团队权限、审计与数据保留策略后再对外开放使用。

## 原理
- 采用 RAG 架构：先检索相关片段，再把上下文注入模型推理。
- 通过 workspace 抽象隔离不同团队或任务的数据边界。
- 将模型路由、知识库管理和会话记录统一在同一平台层。

## 价值
- 可快速落地企业内部知识助手与文档问答。
- 支持多模型切换，便于在成本、速度、质量之间做策略平衡。
- 社区活跃，适合做私有化 AI 中台的起步方案。

## 风险边界
- RAG 质量高度依赖文档清洗、切块策略与索引质量。
- 若权限隔离配置不严，存在跨团队数据泄露风险。
- 生产环境对可用性和备份恢复要求高，不能只按 demo 配置上线。

## 补充建议
- 上线前先做“高频真实问答集”离线评测，量化命中率与幻觉率。
- 建立分级权限与敏感字段脱敏流程。
- 为索引构建、增量更新和异常恢复准备自动化脚本。
