# MOSS-TTS

## 项目定位
MOSS-TTS 是 OpenMOSS 发布的开源语音/声音生成模型家族，覆盖长文本语音、多说话人对话、角色音色与实时流式 TTS 场景。

- GitHub: https://github.com/OpenMOSS/MOSS-TTS

## 用法
- 按仓库说明选择对应子模型与模型卡（model card）进行推理与部署。
- 典型场景包括：有声内容生成、对话系统语音输出、角色化语音设计、环境音生成。

## 原理
- 通过生成式语音模型把文本条件映射到高保真声学输出，并在表达力与稳定性上做工程优化。
- 以“模型家族”形式覆盖不同复杂度与时延需求，而非单一模型通吃。

## 价值
- 提供可本地化和可定制的开源 TTS 选项，减少对闭源商用 API 的单点依赖。
- 在多说话人和复杂音色场景中具备较高可玩性。
- 适合构建 AI 助手、数字人、音频内容生产流水线。

## 风险边界
- 声音滥用风险：需严格控制语音克隆与身份冒用。
- 法律合规：涉及授权语料、肖像/声音权和地区监管。
- 资源需求：高质量语音模型往往需要较高算力与推理优化。

## 补充建议
- 默认加入水印、来源声明和使用审计。
- 生产环境优先评估时延、稳定性与多语言表现。
- 对“仿真人物语音”设置额外审批流程。
