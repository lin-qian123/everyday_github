# presenton

## 项目定位
presenton 是开源 AI 演示文稿生成器，定位为 Gamma/Beautiful.ai 等在线工具的开源替代方案，支持从输入内容到可用幻灯片的快速生成。

- GitHub: https://github.com/presenton/presenton
- Trending 参考: https://github.com/trending

## 用法
1. 按官方文档安装依赖并启动服务。
2. 输入主题、提纲或文稿素材。
3. 选择模板风格并生成演示稿。
4. 对文案、结构、图示进行人工复核后导出。

## 原理
- 内容理解：将文本拆成“章节-要点-叙事节奏”。
- 版式生成：根据模板约束自动分配标题、正文、图文关系。
- 输出层：支持在线编辑与导出接口，形成可复用生产链路。

## 价值
- 显著降低从“想法”到“可讲述材料”的时间成本。
- 适合内部汇报、技术分享、方案草拟。

## 风险边界
- 自动生成内容可能出现事实错误或过度概括。
- 设计质量受模板与输入质量影响较大。
- 涉及商业敏感材料时需注意隐私与合规。

## 补充建议
- 把 AI 生成定位为“初稿加速器”，关键页必须人工重写。
- 固化企业模板和术语库，提升输出一致性。
- 对外发布前增加事实核验与版权检查清单。
