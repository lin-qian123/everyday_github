# AFFiNE（toeverything/AFFiNE）

## 这是什么

`AFFiNE` 是面向知识工作流的开源工作区工具，融合文档、白板与数据库视图，并逐步叠加 AI 助手能力。它在 AI 工具生态中属于“知识生产 + 协作界面层”。

## 怎么用（最短路径）

### 1) 先用官方在线版体验

- 直接访问官方主页完成核心流程体验（页面、白板、数据库、分享）。

### 2) 本地部署（按官方文档）

```bash
git clone https://github.com/toeverything/AFFiNE.git
cd AFFiNE
```

- 按文档执行依赖安装与启动。

### 3) 做一个真实场景验证

1. 新建“项目知识库”并串联文档与看板。
2. 用 AI 功能做摘要/改写。
3. 检查多人协作与权限行为。

## 核心原理

1. 块编辑器 + 多视图模型：同一内容在文档/表格/看板间切换。
2. 本地优先与同步能力：兼顾离线体验与团队协作。
3. AI 增强层：在知识工作流中提供生成、整理与检索辅助。

## 适用场景

1. 研发/产品团队项目知识沉淀。
2. 内容生产与资料整理。
3. 个人知识库与任务管理融合。

## 价值与风险

价值：
- 把多种工作形态整合在同一工作区。
- 适合作为团队知识运营入口。

风险：
- 全量迁移旧系统存在成本。
- AI 生成功能需要二次校验，避免知识污染。

## 我认为有价值的补充材料

1. 先从单团队试点，建立内容模板和命名规范。
2. 明确 AI 生成内容的审核责任人。
3. 定期做知识库“冷数据清理”，防止协作噪声。
