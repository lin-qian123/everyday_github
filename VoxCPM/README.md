# VoxCPM

## 项目定位
`VoxCPM` 是 OpenBMB 发布的多语种语音生成/克隆方向项目，强调 tokenizer-free 的语音建模路线，目标是提升语音自然度与可控性。

- GitHub：https://github.com/OpenBMB/VoxCPM
- 适用人群：语音产品团队、TTS 研究者、多语种内容创作团队。

## 用法

1. 按官方环境说明安装依赖（GPU/CUDA/推理框架）。
2. 下载或准备模型权重与示例音频。
3. 运行推理脚本完成语音合成、风格迁移或克隆实验。

建议先从官方 demo 和短音频样例开始，逐步扩展到业务数据。

## 原理

- 核心路线：弱化传统 tokenizer 依赖，直接在语音特征空间建模。
- 训练目标：兼顾可懂度、韵律自然度和说话人一致性。
- 推理链路：文本/提示 -> 声学表征 -> 波形合成。

## 价值

- 多语种和创意语音设计空间更大。
- 对内容出海、配音自动化有直接生产力价值。
- 为端到端语音模型提供新工程范式参考。

## 风险边界

- 声音克隆涉及明确授权与可追溯水印要求。
- 训练/推理成本较高，需做资源预算。
- 真实业务中需额外建设滥用检测与申诉流程。

## 补充建议

- 先做“授权样本 + 白名单场景”的小规模试点。
- 在输出链路增加 AI 音频标识与内容溯源信息。
- 对外发布前评估不同语种下的误读/口音偏差。
