# Deep-Live-Cam

一款实时换脸与一键视频 deepfake 工具：只需一张源人脸图片即可对视频、图片或摄像头画面进行替换。面向内容创作、角色动画与直播场景，强调“低门槛 + 实时 + 本地运行”。

## 它解决什么问题
- 传统换脸流程复杂、需要多张素材或长时间训练；Deep-Live-Cam 只需要单张人脸即可完成实时或批处理换脸。
- 对直播/视频通话等低延迟场景，提供“实时预览 + 可选增强”的管线。
- 对隐私敏感的场景，尽量做到本地离线处理（不上传素材）。

## 核心原理（简要）
- **人脸分析与检测**：识别并定位目标画面中的人脸区域。
- **人脸替换**：用 `inswapper_128_fp16.onnx` 模型把源人脸特征映射到目标脸上。
- **画质增强**：可选 `GFPGAN` 模型对替换后的人脸进行修复/增强。
- **多执行后端**：支持 CPU、CUDA、CoreML、DirectML、OpenVINO 等执行提供程序，便于在不同硬件上加速。

## 使用方式（高层流程）
1. 选择一张源人脸图片（Source）。
2. 选择目标：图片 / 视频 / 摄像头（Target）。
3. 选择处理模式与可选增强（如 face enhancer、mouth mask）。
4. 一键开始：实时预览或输出处理后的视频/图片。
