# CodexBar

概览
CodexBar 是一款 macOS 状态栏应用，用于快速打开与管理 OpenAI Codex 工具。它提供全局快捷键、菜单栏入口与自动切换等能力，便于在编码过程中快速呼出 Codex 相关功能。

核心能力
- 状态栏常驻入口：无需切换窗口即可呼出工具。
- 快捷键支持：快速打开/切换 Codex。
- 自动切换与轻量控制：适合高频使用场景。

原理与架构（基于公开说明的归纳）
CodexBar 以 macOS 菜单栏应用的形式运行，将 Codex 相关入口与快捷键绑定到系统层，减少上下文切换成本。

快速上手
- 安装应用后授予必要的系统权限。
- 使用菜单栏入口或快捷键快速打开 Codex。

使用场景
- 需要频繁调用 Codex 的开发者工作流。
- 多项目切换时减少打开窗口与定位工具的时间。
