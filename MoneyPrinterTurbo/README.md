# MoneyPrinterTurbo

## 项目定位
MoneyPrinterTurbo 是一个面向短视频生产的自动化工具，核心目标是用大模型与模板流水线快速生成可发布的视频内容。

- GitHub: https://github.com/harry0703/MoneyPrinterTurbo

## 用法

### 1. 本地部署
- 按仓库文档安装依赖并配置模型与素材参数。
- 通过 Web UI 或命令行输入主题、风格、时长等信息。

### 2. 生成流程
- 输入选题/脚本要点。
- 生成文案与分镜。
- 合成语音、画面与字幕。
- 导出成片并二次剪辑。

## 原理
- 通过 LLM 生成脚本与结构化分镜。
- 利用 TTS、素材检索/拼接、字幕对齐等模块形成自动化流水线。
- 通过模板化参数（语速、镜头节奏、标题风格）提高批量生产效率。

## 价值
- 降低短视频从想法到初版成片的时间成本。
- 对内容团队和个体创作者都具备较高的 MVP 验证价值。
- 适合做多主题 A/B 测试与批量选题实验。

## 风险边界
- 自动生成内容可能出现事实错误或表达同质化。
- 素材版权、配音授权与平台合规审查需要单独治理。
- 批量自动化发布若缺乏人工复核，容易放大内容风险。

## 补充建议
- 将该工具定位为“初稿工厂”，发布前必须人工复审。
- 建立选题白名单、敏感词规则与事实校验流程。
- 对外发布前补充“脚本来源、素材来源、授权状态”检查项。
