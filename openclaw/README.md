# OpenClaw (openclaw/openclaw)

概览
OpenClaw 是一款本地优先的个人 AI 助手与代理平台，强调“运行在你的设备上、通过你熟悉的聊天应用完成真实任务”。它支持多种聊天渠道接入（如 WhatsApp/Telegram/Slack/Discord 等），并提供持久化记忆、浏览器控制与插件扩展等能力。官方提供一键安装脚本与本地仪表盘，强调隐私与自托管安全。

核心能力
- 本地优先与自托管：数据尽量保留在本地设备或自建环境中，强调隐私与可控性。
- 多渠道聊天接入：可在常用聊天应用内与助手交互并触发行动。
- 持久化记忆：跨会话保存上下文与偏好。
- 插件/技能扩展：可通过社区技能或自定义插件扩展能力。

原理与架构（基于公开说明的归纳）
OpenClaw 的工作模式是“聊天入口 + 本地执行 + 可扩展技能”。你在聊天应用里发出任务，OpenClaw 通过本地运行的控制平面调用相应工具或插件执行，再回写结果。它强调本地运行和最小化外部暴露，从而降低数据外泄与权限滥用风险。

快速上手
- 安装与引导：官方提供一键安装脚本，安装后运行引导向导，绑定聊天通道并配置模型。
- 典型流程：安装 -> 启动向导 -> 连接模型与通道 -> 在聊天应用中发送任务。

使用场景
- 个人效率：收件箱清理、日程安排、信息检索等。
- 自动化与集成：通过插件扩展与聊天入口，实现跨工具的轻量自动化。

风险与注意事项
- 需谨慎授予系统与账户权限，避免过度授权带来的安全风险。官方文档强调对通道与网关暴露的安全配置。
